Overview

This tool augments PubMed search to mirror human iteration, including...

  • Multi-strategy query generation for comprehensive coverage
  • Filtered results for relevance to original query
  • Content analysis and synthesis via large language models
  • Automated citation validation and correction, no more hallucination

Search Features

  • Natural Language Queries: Describe your research question in plain English
  • Advanced Filters: Use the gear icon to filter by study type, publication date, and result count
  • Smart Query Generation: Automagically broaden/narrow/optimize PubMed search strategies

Analytic Features

  • Relevance Ranking: Results re-ranked based on content relevance to your query
  • Literature Synthesis: Generates comprehensive summaries with proper citations
  • Custom Prompts: Use the lightning bolt (⚡) to customize prompt
  • Citation Validation: Automatically verifies and corrects PMID references

Understanding Results

  • Strategy: Shows which search strategy found each paper
  • IF (Impact Factor): Journal impact factor when available
  • Citations/Year: Average annual citations from iCite database
  • Total Score: Proprietary metric including recency, citations, and impact factor
  • AI Rank: AI-determined relevance ranking (appears after analysis)

Export Options

  • Export RIS: Download all results in RIS format for reference managers
  • Download Synthesis: Get both text summary and RIS file for cited papers
  • Hover Citations: Hover over PMID links in synthesis for paper details

Advanced Tips

  • Sorting: Click sort buttons to reorder by date, citations, AI rank, or total score
  • Custom Analysis: Use the lightning bolt to custom-prompt synthesis structure and requirements
